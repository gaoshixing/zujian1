
## 院校库模块

```
|____router.js                //此模块路由
|____src                     //源码  
| |____assets                //静态资源   
| | |____svg                 //svg图标
| |____libs                  //工具方法
| | |____request.js          //所有请求接口工具类
| |____modules               //自定义组件
| |____views                //页面
```

### 路由定义

院校库模块顶级路由         `spoc-library`

学校管理                    `library.schoolManage`

学院管理                    `library.academeManage`

标准专业库                   `library.optionalLibrary`

藤门搜索                      `library.search`
